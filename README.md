Descripción del proyecto
El proyecto propone un modelo de inteligencia artificial para la detección de anomalías en métricas de infraestructura tecnológica, utilizando técnicas de aprendizaje automático no supervisado.
El modelo analiza métricas de servidores como CPU, memoria y uso de disco con el objetivo de identificar comportamientos anómalos que puedan indicar fallas operativas o incidentes en los sistemas.
