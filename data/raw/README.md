# Data - Raw

Esta carpeta contiene los datos originales utilizados en el proyecto.

⚠️ Nota:
Los datos reales no se incluyen en este repositorio debido a políticas de confidencialidad de la organización.

## Descripción de los datos

Los datasets corresponden a métricas de infraestructura TI, tales como:

- Uso de CPU
- Uso de memoria
- Actividad de disco
- Métricas de red

## Formato esperado

Archivo CSV con las siguientes columnas:

- timestamp_utc
- instance
- metric
- value

## Frecuencia de datos

Los datos son recolectados en intervalos de 15 minutos.

## Fuente

Herramientas de monitoreo como Prometheus.
